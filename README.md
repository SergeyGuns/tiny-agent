# 🤖 Minimalist Multi-Agent Development Framework

Минималистичный фреймворк для создания автономных ИИ-разработчиков на базе TypeScript. Архитектура построена на чистом протоколе **ReAct (Reason + Act)** и паттерне **Actor-Critic (Разработчик + Супервизор)** без использования тяжелых сторонних абстракций (LangChain / LlamaIndex). 

---

## 🎯 Основная концепция

Проект доказывает, что для создания надежного ИИ-агента достаточно стандартной библиотеки Node.js и прямых запросов к API модели. Система спроектирована для безопасного написания кода, автоматического тестирования и самокоррекции с жестким контролем изменений через Git.

### Философия разработки:
* **Zero-Dependency Core:** Минимальное количество внешних зависимостей. Только нативный `fetch`.
* **Полная прозрачность:** Каждое действие и мысль логируются в файл сессии.
* **Безопасность через транзакции:** Любое действие агента может быть автоматически откачено, если оно ломает проект.

---

## 🏗️ Архитектура и Жизненный цикл

Система разделяет обязанности между двумя независимыми агентами: **Developer Agent** (исполнитель) и **Supervisor Agent** (контролер качества).

[Старт] ──> Инициализация ──> Планирование ──> Исполнение
                                                  │
[Финал] <── Вердикт <─────── Анализ <───────── Логирование
---

## ✨ Ключевые возможности

* **Режим планирования:** Агент обновляет итеративный план перед каждым действием. Это снижает хаотичность вызовов.
* **JSON-аргументы:** Защита от ошибок парсинга при передаче длинного кода или путей.
* **Текстовый TUI:** Пошаговая цветная визуализация работы в терминале через ANSI-коды.
* **Автономный логгер:** Полная запись сессии в `session.log` для отладки и аудита.
* **Авто-откат (Rollback):** Защита рабочей директории с помощью автоматических команд Git.

## 📦 Быстрый старт

### 1. Требования
* Node.js v18 или выше.
* Установленный Git в рабочей директории.

### 2. Установка
Склонируйте репозиторий и установите типы для разработки:
```bash
npm install
```

### 3. Настройка окружения
Создайте переменные окружения для доступа к моделям и поиску:
```bash
export OPENAI_API_KEY="ваш_ключ_openai"
export SERPER_API_KEY="ваш_ключ_serper_для_web_search"
```

### 4. Запуск
Запустите интерактивный TUI-интерфейс:
```bash
npx tsc && node dist/agent.js
```

---

## 🛠️ Доступные инструменты агента

1. `webSearch` — Поиск актуальной информации в сети интернет.
2. `readDir` — Просмотр структуры файлов в изолированном воркспейсе.
3. `readFile` — Чтение исходного кода файлов.
4. `writeFile` — Безопасная запись сгенерированного кода.
5. `runPackageScript` — Запуск тестов, линтеров и сборки из `package.json`.
